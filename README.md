# Anti-Pay-To-Win-Distributing-Mod

Youtube Channel: https://www.youtube.com/@Sapire

What Each Command Does:

/PayAll {value} - equally pays everyone - {value} is the ammount of money you have

------------------------------------------------------------------------------
# How to Buy Out The Auction House Correctly

IMORTANT: Use SetUpAuctionHouseBuying First then BuyOutAuctionHouse

1) /SetUpAuctionHouseBuying {First value} {Second value} - Sets up buying the Auction house

First value is the position of the items 

Second value is the position of the buy button

To Find the the position of the item or the position of the buy button please go to the buttom of the GitHub Page to see the corisponding number with the position in the AH

2) /BuyOutAuctionHouse - Starts to buy out the auction house

If your inventory is full it will stop trying to buy out the Auction house

If the same item has tried to be bought 4 times it will stop

------------------------------------------------------------------------------

/STOPWHATDOING - Stops "PayAll" and "BuyOutAuctionHouse"

/ChangeTicks {value} - changes the ammount of time the client waits till it pays the next person or buys the next item. {value} is the amount of ticks till the next command
